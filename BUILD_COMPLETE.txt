EXAMPLE: SEND NOTIFICATION WITH ACTIONS

curl -X POST https://v0-push-notification-api.vercel.app/api/send \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer sk_yourkey..." \
  -d '{
    "subscriptionId": "sub_abc123",
    "notification": {
      "title": "Order Shipped!",
      "body": "Your order #12345 is on the way",
      "image": "https://example.com/shipped.jpg",
      "icon": "/order-icon.png",
      "actions": [
        { "id": "view", "title": "View Order" },
        { "id": "track", "title": "Track Package" }
      ],
      "data": {
        "orderId": "12345",
        "actionUrls": {
          "view": "https://yoursite.com/orders/12345",
          "track": "https://track.courier.com/12345"
        }
      }
    }
  }'

Response:
{
  "success": true,
  "message": "Sent 1 notification(s)",
  "result": { "successful": 1, "failed": 0, "errors": [] }
}
