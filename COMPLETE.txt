📝 SENDING NOTIFICATIONS

Via Web UI:
- Visit http://localhost:3000
- Click Subscribe
- Fill in title & message
- Click "Send Test Notification"

Via cURL:
curl -X POST http://localhost:3000/api/send \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer test-api-key-12345" \
  -d '{
    "subscriptionId": "sub_abc123",
    "notification": {
      "title": "Hello",
      "body": "Your message here"
    }
  }'

Via JavaScript:
fetch('/api/send', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer test-api-key-12345'
  },
  body: JSON.stringify({
    subscriptionId: 'sub_abc123',
    notification: { title: 'Hello', body: 'Message' }
  })
})

🔄 WORKFLOW

1. User subscribes device:
   Browser → Service Worker → /api/subscribe → Firebase RTDB ✓

2. Admin sends notification:
   API request → /api/send → Web Push → User's browser ✓

3. Notification received:
   Service Worker handles push event → Shows notification ✓

4. User clicks notification:
   Service Worker handles click → Opens URL (customizable) ✓
