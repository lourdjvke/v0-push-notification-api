4. API Reference

Key endpoints and examples:

Subscribe Device (No API Key):

POST /api/subscribe Headers: Content-Type: application/json Body: { subscription: {...}, deviceName: "..." } Response: { subscriptionId: "sub_..." }

Send Notification (Requires API Key):

POST /api/send Headers: - Content-Type: application/json - Authorization: Bearer Body: { subscriptionId: "sub_...", notification: { title, body, image, actions, ... } } Response: { success: true, result: { successful, failed } }

Unsubscribe (No API Key):

DELETE /api/unsubscribe?id= Response: { success: true }
